# Wind Check


# Explanation
Asks user for zip code, then checks NOAA's weekly forecasts for wind greater than 5mph.
 
 
# Requirements
Packages used:
- noaa_sdk
- pandas
- statistics

 
# Installation
- Python 3
- Pandas
- noaa_sdk
